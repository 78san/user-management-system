# User Management Project

## Project Description
This is a simple **User Management Project** built using **Python** and **MySQL**.  
It allows users to **add, update, delete, and view users** in the database with a secure login system.  
The project demonstrates basic CRUD operations and how to connect Python with a MySQL database.

## How to Run / Use
1. **Requirements / Dependencies**
   - Python 3.x
   - MySQL
   - `mysql-connector-python` library (install using `pip install mysql-connector-python`)

2. **Setup**
   - Clone the repository:
     ```
     git clone https://github.com/sanjayarya/UserManagementProject.git
     ```
   - Open the project folder in your IDE (VS Code, PyCharm, etc.)
   - Configure database credentials in `config.py` (or wherever your DB connection is defined)

3. **Run the Project**
   - Open terminal / command prompt in project folder
   - Run the main Python file:
     ```
     python main.py
     ```
   - Follow on-screen instructions to use the application

## Features
- Add new users
- Update existing users
- Delete users
- View all users
- Secure login system
